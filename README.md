# Roblox Universal UI

一个模块化的 Roblox 通用 UI 脚本框架，集成多种实用功能。

## 功能

- **无头** - 隐藏角色头部和脸部
- **断腿** - R6/R15 角色断腿效果
- **画质简化** - 降低画质提升性能
- **隐藏饰品** - 按类型独立控制8种饰品可见性
- **灵敏度调节** - 触摸灵敏度 0.1-10倍可调
- **FFlag工具** - JSON粘贴、保存、加载、应用
- **FPS监控** - 实时显示FPS和PING
- **记忆功能** - 面板尺寸和位置自动保存
- **Toast提示** - 操作反馈通知

## 使用方法

1. 将所有文件放入 Roblox 脚本执行器中
2. 运行 `src/Main.lua`
3. 点击屏幕上的 FPS 显示器打开控制面板
4. 拖动 FPS 显示器和面板标题栏移动位置
